MedStroke AI is a browser-based project for brain CT review. It combines classification, segmentation, ensemble reasoning, and explainability outputs to help present a structured technical screening result before a user enters the analyzer workspace.

What The Website Delivers

One workflow from image upload to precautions.

  • Upload a CT image in common image formats

Technical Core

The platform uses trained checkpoints stored in project artifacts and renders inference outputs directly in the browser for fast testing and presentation.

How It Works

From CT image to structured screening output

01

Upload a CT Image

The user selects a brain CT image in PNG, JPG, JPEG, or BMP format.

02

Run Trained Models

The system loads the selected AI model, stroke priors, and ensemble logic for inference.

03

Generate Visual Evidence

It creates classification charts, segmentation overlays, GradCAM views, and metrics panels.

04

Show Precautions

The report includes urgency guidance, response steps, and practical precautions for review.

Website Features

Visual Explainability

Shows GradCAM and segmentation outputs so the result is easier to inspect visually.

Precautions and Response

Provides precaution lists and immediate response steps based on the predicted result.

Outputs You Will See

The analyzer presents more than a single label

Classification Board

Probability charts make the predicted class easier to compare and explain.

Segmentation View

Stroke-prior overlays help visualize the region the model focuses on for lesion-aware review.

GradCAM Evidence

Model Comparison

Cross-model outputs show where multiple models agree or differ on the same scan.

Precaution Notes

Each generated result is paired with response steps and basic safety-oriented guidance.

Downloadable Report

All key findings can be bundled into a downloadable project report from the browser.

Important Note

This website is a technical screening interface, not a medical diagnosis.

The project is designed to present AI model outputs, visual evidence, and precaution guidance. If stroke symptoms are active, emergency medical care should always come first.
